The Errant Analyst

description of chart legend items:

BL - the benchmark specific r-squared is calculated using a linear regression with polynomial features, generating a baseline metric (BL).

A_ALL - regression including a daily mean sentiment score for all comments (A_ALL) containing the response variable ticker (AAPL) or common name (apple) in the comment text.

PCA_# - regression similar to the A_ALL process except that comments for this metric are a subset of all ticker/name comments. The subset is produced by an unsupervised clustering process (PCA_#) is applied to surface comments with pronounced stock-trading idea language.
